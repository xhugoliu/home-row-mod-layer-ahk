# home-row-mod-layer-ahk

An AutoHotkey v1 script for using a laptop keyboard like a reduced version of my split keyboard workflow.

The script focuses on the parts I use most from my ZMK keymap:

- home-row mods on `s d f g h j k l`
- `Space` as a navigation layer when held
- `e + r` chord for `Tab`
- `f + j` chord for `F20`

The original keyboard layout and behavior live here:

- https://github.com/xhugoliu/zmk-config-pskeeb5

## Requirements

- Windows
- AutoHotkey v1.1

## Usage

Run `home_row_mod_layer.ahk`.

Default timing values:

- tap-hold threshold: `180 ms`
- chord threshold: `50 ms`

## Notes

This is a laptop-keyboard adaptation, not a full 1:1 port of the complete ZMK keymap.
